# Housing Data Analysis

## 🏠 Housing Data Analysis Dashboard

This project analyzes housing data using Python and Streamlit to understand trends, pricing, and patterns in the real estate market.

---

## 📊 Project Overview

The goal of this project is to:

- Analyze housing dataset for key insights
- Find patterns in prices based on location, size, and features
- Perform data cleaning and preprocessing 
- Generate interactive visualizations using Python
- Build a Streamlit dashboard for easy data exploration

---

## 🛠️ Tools & Technologies

- Python 🐍
- Pandas
- Streamlit
- Matplotlib / Seaborn
- Git & GitHub

---

## 📁 Project Files

- `streamlit_deebiga_dashboard.py` → Main Streamlit dashboard app
- `batch 4 dataset-housing data (1).csv` → Housing dataset 
- `report of housing data.pdf → Project report PDF
---

## ▶️ How to Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/deebiga27/Housing-data-analysis.git
   ```

2. **Install dependencies**:
   ```bash
 pip install streamlit pandas matplotlib
   ```

3. **Run the app**:
   ```bash
 py -m streamlit run streamlit_deebiga_dashboard.py
   ```
